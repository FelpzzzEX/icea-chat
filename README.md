# **CSI606-2025-01 - Remoto - Proposta de Trabalho Final**

## *Discente: Felipe Fialho (21.1.8166)*

### Resumo

Será um sistema inspirado em fóruns de bate-papo antigos que dominavam a internet, onde os usuários poderão criar contas, adicionar amigos e interagir entre si, seja por conversas privadas ou em grupos. O sistema terá uma interface simples e moderna, permitindo comunicação em tempo real entre usuários.

---

### 1. Tema

O trabalho final tem como tema o desenvolvimento de uma aplicação web de chat, inspirada em fóruns de conversas antigos, porém com abordagem modernizada. A plataforma permitirá criação de perfis, adição de amigos, criação de grupos e troca de mensagens em tempo real, tanto individualmente quanto em grupos.

---

### 2. Escopo

Este projeto terá as seguintes funcionalidades:

**Para usuários gerais:**

* Criar conta e realizar login.
* Editar perfil.
* Adicionar e excluir amigos.
* Conversar via chat (privado ou em grupo).
* Excluir conta e realizar logout.

**Para moderadores:**

* Apagar mensagens.
* Expulsar membros de grupos.
* Conceder cargos a outros usuários.

---

### 3. Restrições

Neste trabalho **não serão considerados**:

* Autenticação avançada (2FA, login por redes sociais, recuperação de senha por e-mail).
* Upload de arquivos/envio de mídia (apenas mensagens de texto).
* Criptografia ponta a ponta nas conversas.
* Notificações push reais — apenas atualização via WebSocket.
* Aplicativo mobile.
* Sistema de recomendação de amigos/grupos.
* Painel de administração completo com logs detalhados — apenas ferramentas básicas de moderação.

---

### 4. Protótipo

Foram elaborados protótipos das principais páginas da aplicação, incluindo:

* Tela de Login
* Tela de Cadastro
* Tela de Perfil
* Página de Lista de Amigos
